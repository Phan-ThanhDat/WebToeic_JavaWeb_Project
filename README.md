Teaching Toeic Web App
==============

**This repository contains the source code of the application built the web app to teach learner to study English using JSP and Servlet with Model–view–controller **




Below are some instructions how to work with the web application.

Workflow
========

Set up the Tomcat Apache  and run it.

To run the application, run your browser like Chrome  and open http://localhost:8080/ .
